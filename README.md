## 安装步骤

- 克隆到本地

    `
    git clone https://github.com/bdlite/hooks.git
    `

- 安装

    `
    yarn install
    `

- 开始单元测试

    - 可以直接 run (指向的是es目录)

        `
        npm run test
        `

    - 也可以模拟发布前

        `
        npm run prepublishOnly
        `
